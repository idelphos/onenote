CSS3 selector

  1. 标签选择器 h1｛color:red;font-size:12px;}
  2. ID选择器 #idName {color:red;font-size:12px;}
  3. class选择器 .className {color:red;font-size:12px;}

ID 选择器和class选择器比较

ID选择器和class选择器都可以应用于任何元素。

  • ID选择器只能在文档中使用一次，但class无些限制
  • 可以使用类选择器方法为一个元素设置多个样式，ID选择器不可以
.stress{
    color:red;
}
.bigsize{
    font-size:25px;
}
<p>到了<span class="stress bigsize">三年级</span>下学期时，我们班上了一节公开课...</p>
